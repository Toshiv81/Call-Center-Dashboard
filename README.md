📞 Call Center Performance Dashboard
📌 Project Overview

This project is an interactive Call Center Analytics Dashboard built using Power BI. The dashboard provides deep insights into customer support operations, helping businesses monitor call center performance, customer satisfaction, response efficiency, and customer sentiment.

The project analyzes customer interaction data collected from multiple support channels such as:

Call Center
Chatbot
Web Support
Email Support

The dashboard is designed to help stakeholders quickly identify operational bottlenecks, improve customer experience, and make data-driven decisions.

🚀 Objectives

The main goals of this dashboard are:

Track overall call center performance
Analyze customer satisfaction (CSAT)
Monitor SLA compliance
Understand customer sentiment trends
Compare support channels performance
Identify high-volume issue categories
Analyze average call duration
Monitor regional/state-wise performance

📈 Key Insights

Based on the dashboard analysis of 32,941 customer interactions, the following business insights were identified:

📞 Total customer interactions analyzed: 32,941
⭐ Average CSAT Score: 5.55 / 10
⏱️ Average Call Duration: 25.02 minutes
✅ Calls resolved within SLA: 62.1%
❌ Calls above SLA: 37.9%

📌 Major Findings
Negative and very negative sentiments together account for more than 51% of all customer interactions.
SLA compliance has a direct impact on customer satisfaction scores.
Billing Questions and Service Outages generate the highest support volume.
Call Center and Chatbot channels handle the majority of customer requests.
Higher call duration is often associated with lower CSAT ratings.
Certain regions show consistently higher complaint volumes, indicating operational improvement opportunities.
